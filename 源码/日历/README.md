# dateMarker
签到日历


在普通日历的基础上，添加自定义的标签。

#### 用法示例
    $('.date').dateTags({
        'tags':[
            {'time':'2015/1/3','tag':'考试报名'},
            {'time':'2015/2/1','tag':'申报'},
            {'time':'2015/2/16','tag':'网上缴费'},
            {'time':'2015/3/11','tag':'网上缴费'},
            {'time':'2015/5/26','tag':'网上缴费'},
            {'time':'2015/6/6','tag':'网上缴费'}
        ]
    });
